## Endpoints
> This section will grow as features are added.

### Endpoints List

### SAMPLE
- `GET /` ‚Üí Welcome message
- `GET /sample/` ‚Üí Sample JSON response

### DAY 1 ENDPOINTS
- `POST /day1/items` ‚Üí Create a new item

- `GET /day1/items` ‚Üí Get all items

- `GET /day1/items/{item_id}` ‚Üí Get item by ID

- `PUT /day1/items/{item_id}` ‚Üí Update item by ID

- `DELETE /day1/items/{item_id}` ‚Üí Delete item by ID

### DAY 2 ENDPOINTS
- `POST /day2/token` ‚Üí Generate JWT access token using username & password

- `GET /day2/secure-data` ‚Üí Access secure data (requires valid Bearer token in header)
